## Desafio Técnico do IATECAM
## Analista de Testes

**Instruções:**

O projeto do Cypress encontra-se já configurado no diretório "3-automation". Foi necessário adicionar `chromeWebSecurity: false` ao arquivo de configuração `cypress.config` para conseguir abrir o site pelo Cypress. Por isso, só é possível rodar os specs em navegadores Chromium. Por exemplo, consegui rodá-los com sucesso no Edge e no Electron.
